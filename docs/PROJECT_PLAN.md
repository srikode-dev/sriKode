# SriKode - Project Plan

## Project Overview

SriKode is a developer-focused platform that provides:

* Web Development Tutorials
* HTML/CSS Components
* JavaScript Projects
* React Projects
* MERN Stack Projects
* Technical Blogs
* Source Code Repositories
* YouTube Integration

---

# Tech Stack

## Public Website

* Next.js
* JavaScript
* Tailwind CSS
* App Router
* SEO Optimized

## Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Cloudinary

## Admin Panel

* React
* Vite
* Tailwind CSS
